# 广度优先遍历

广度优先遍历类似于一个分层搜索的过程，广度优先遍历需要使用一个队列以保持访问过的结点的顺序，以便按这个顺序来访问这些结点的邻接结点。

具体算法表述如下：

- 访问初始结点v并标记结点v为已访问。
- 结点v入队列
- 当队列非空时，继续执行，否则算法结束。
- 出队列，取得队头结点u。
- 查找结点u的第一个邻接结点w。
- 若结点u的邻接结点w不存在，则转到步骤3；否则循环执行以下三个步骤：<br> 
	(1). 若结点w尚未被访问，则访问结点w并标记为已访问。<br>
	(2). 结点w入队列<br>
	(3). 查找结点u的继w邻接结点后的下一个邻接结点w，转到步骤6。
